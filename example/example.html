Favicon Dart

A Dartlang library for creating animations and badges for the favicon icon.

Click the buttons above to view a quick demo. More demos provided below.

Badge with no animation

Favicon f = new Favicon(); 
Badge b = new Badge(); // Create a badge element
f.addElement(b); // Add the badge element to the favicon
b.badge(1); // Update the badge number in the favicon

Badge with fade animation

Favicon f = new Favicon(); 
Badge b = new Badge(); // Create a badge element
f.addElement(b); // Add the badge element to the favicon
b.fadeInBadge(1); // Fades in the badge number

Badge which immediately updates rather than queues animations

Favicon f = new Favicon(); 
Badge b = new Badge(); // Create a badge element
f.addElement(b); // Add the badge element to the favicon
b.slideInBadge(1, "up", startInstantly: true); // Slides in the badge number instantly
